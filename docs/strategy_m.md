# 策略M: CSI800 低换手+动量 个股因子选股

## 概述

从中证800（沪深300+中证500）**历史真实成分股**中，使用低换手率+12个月动量双因子选股，Top20等权配置，月度调仓。

这是整个策略体系中**Sharpe最高**的策略（0.97），10年回测CAGR达21.4%。

## 核心逻辑

### 选股流程

1. **宇宙**: 每月底从当时CSI800真实成分股中选股（baostock历史成分股API）
2. **因子计算**:
   - **低换手率**: 20日平均换手率，rank升序（换手率低的排前面）
   - **12M动量(skip1M)**: 过去12个月收益率跳过最近1个月，rank降序（动量高的排前面）
3. **多因子合成**: 等权合成 composite = 50% * rank_turn + 50% * rank_mom
4. **选股**: composite最低的Top20只入选
5. **配置**: 等权买入，持有一个月，下月底调仓

### 子策略

| 子策略 | 因子组合 | Top N | CAGR | Sharpe | MDD |

M-A为主策略，M-B加入EP（盈利收益率）因子作为价值增强变体。

## 因子解读

### 为什么低换手+动量有效？

- **低换手率**: 筛除高换手的热门投机股，选出被市场忽视的冷门股。这些股票往往估值更合理、波动更低。
- **12M动量(skip1M)**: 学术界经典动量因子，跳过最近1个月是为了规避短期反转效应。捕捉中期趋势延续。
- **组合效果**: 低换手过滤掉噪声交易密集的股票，动量捕捉趋势，两者互补形成高质量动量策略。

## 注意事项

- 月度调仓，每月持有约20只个股，换手率相对较高
- 单边30bps交易成本已为保守估计（含冲击成本）
- 2017年和2020年贡献了大部分超额收益，近年（2022-2025）表现平淡
- 个股策略执行难度高于ETF策略，需考虑流动性和下单效率
