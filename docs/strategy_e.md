# Strategy E: 逆向价值+质量策略

> 在龙头中寻找近期被低估的优质股 · 与策略D形成动量/价值互补

## 1. 策略概述

**核心思想**：从28只龙头股中，寻找近期表现较差（=便宜）但波动率低（=质量高）的股票，即"跌得多但跌得稳"的优质龙头。逆向投资思路，与策略D的动量策略形成天然互补。

| 维度 | 内容 |
|------|------|
| 标的 | 28只A股行业龙头股（同策略D） |
| 信号频率 | 月度（每4周运行一次） |
| 选股逻辑 | 10周动量反转（逆向排名）+ 低波动质量因子 |
| 安全过滤 | 4周跌幅 > -20% 的排除（防暴跌） |
| 行业约束 | sector_max=1（每行业最多1只） |
| 持仓数 | Top 4，等权25% |
| 交易成本 | 单边8bp |
| 无前瞻偏差 | T周决策 → T+1周收益 |

## 2. 股票池

与策略D共用同一个28只龙头股池（详见 [Strategy D 文档](strategy_d.md#2-股票池28只龙头股)）。

## 3. 策略逻辑详解

### 3.1 价值因子：动量反转

```
价值得分 = -(10周涨幅)
```

- 10周涨幅为**负**（即近期下跌的股票），价值得分为**正**
- 逻辑：龙头股的下跌往往是短期情绪导致，基本面稳固，跌多了就是"便宜"
- 这是**逆向投资**（contrarian）的核心思路

### 3.2 质量因子：低波动

```
质量得分 = -(12周收益率标准差)
```

- 波动率越低，质量得分越高
- 逻辑：低波动的龙头说明市场分歧小、经营稳定

### 3.3 综合排名

```
综合得分 = 0.8 × 价值排名 + 0.2 × 质量排名
```

- 价值权重80%，质量权重20%
- 按综合得分从小到大排名（越小越好）
- 优化发现VW=0.2是sweet spot（VW=0.3在walk-forward中不如0.2稳定）

### 3.4 安全过滤

```python
if 4周动量 < -20%:
    排除该股票  # 暴跌可能是基本面恶化
```

- 排除4周跌幅超过20%的股票
- 防止买入正在"价值陷阱"中的股票
- DD=-0.20比-0.15（更严格）更稳定

### 3.5 行业约束 + 选股

与策略D相同，`sector_max=1` 确保每行业最多1只。从综合排名中自上而下选4只。

### 3.6 调仓频率

每4周（月度）调仓一次，低于策略D的双周频率，因为价值回归需要更长时间。

## 5. 优化历程

与策略D同步进行5轮优化：

| 轮次 | D+E共同发现 |
|------|------------|
| Round 1 | 基础策略开发，72组E参数网格搜索 |
| Round 2 | sector_max=1是最大改进；reversal比mean_reversion类型更稳 |
| Round 3 | 1800 configs精细网格：VLB=8,VW=0.3是Calmar最高 |
| Round 4 | Walk-forward 5折：reversal类型（E4）avg Calmar=3.11，最稳定 |
| Round 5 | 参数灵敏度确认：VW=0.2在WF中优于0.3 |

**关键发现**：
1. `reversal`（逆向）比`mean_reversion`（均值回归）在walk-forward中更稳健
2. `VW=0.2`是sweet spot，比0.3在WF中更好
3. `DD_filter=-0.20`比-0.15宽松一些但更稳定
4. Vol scaling和Regime对个股策略**有害**

## 9. 注意事项

1. **逆向投资风险**：买的是近期下跌的股票，可能继续下跌
2. **安全过滤很重要**：DD_filter=-20%防止买入暴跌股，不可去除
3. **sector_max=1是关键**：与策略D一样，行业分散是回测表现的核心保障
4. **月度调仓**：比策略D低频，因为价值回归需要更长周期
5. **与策略D搭配使用**：两者零重叠、低相关，组合风险收益比优于单策略

## 10. 策略理念总结

```
策略E 的投资哲学：

"龙头不死，跌多了就是机会。"

选的是各行业绝对龙头（茅台、宁德、比亚迪级别），
它们的下跌大概率是市场情绪/行业轮动导致，而非基本面恶化。
用低波动做质量筛选，排除暴跌股做安全保障，
在逆向中寻找确定性。
```
