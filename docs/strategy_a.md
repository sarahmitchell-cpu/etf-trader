# Strategy A: 债券分散持有策略

> 4只债券ETF等权分散 + 季度再平衡 -- 低风险防守底仓

## 1. 策略概述

**核心思想**: 在4只覆盖不同久期和信用等级的债券ETF中等权分散持有，每季度再平衡。这是一个纯被动策略，不做任何择时判断，目标是获取债券市场平均收益并通过分散化降低单一品种风险。

| 维度 | 内容 |
|------|------|
| 策略类型 | 被动持有 + 定期再平衡 |
| 标的 | 4只债券ETF (利率债+信用债+城投债+短债) |
| 调仓频率 | 季度 (每季度末) |
| 配置方式 | 等权 (各25%) |
| 交易成本 | 单边5bp |
| 数据源 | 中证指数公司 (CSIndex API) |

## 2. 标的池

| 指数 | ETF | 代码 | 定位 | 风险特征 |
|------|-----|------|------|---------|
| 十年国债 (H11009) | 国泰十年国债ETF | 511260 | 长久期利率债 | 久期长，利率敏感 |
| 城投债 (H11015) | 海富通城投债ETF | 511220 | 中久期信用债 | 信用利差+久期 |
| 信用债 (H11073) | 平安公司债ETF | 511030 | 中短久期信用债 | 信用利差为主 |
| 短债 (H11006) | 海富通短融ETF | 511360 | 短久期现金替代 | 近似货基，极低波动 |

**分散逻辑**: 四只ETF覆盖利率债/信用债两大类，长/中/短三个久期段。当利率上升时，长久期国债受损最大但信用利差可能收窄；经济下行时信用债受损但利率债受益。四者互补。

## 3. 回测表现

**回测期**: ~15年 (使用中证债券指数，含票息收入)

| 指标 | 数值 |
|------|------|
| CAGR | 4.1% |
| Max Drawdown | -2.1% |
| Sharpe | 1.045 |
| Calmar | 1.97 |

### 年度收益特征
- 极度稳定，年化波动率极低
- 仅在利率快速上行年份（如2013年钱荒、2017年去杠杆）出现小幅回撤
- 适合作为策略组合中的"压舱石"

## 4. 策略逻辑

```
每季度末:
  1. 计算当前4只ETF的持仓比例
  2. 若任何一只偏离25%超过阈值 -> 再平衡至等权
  3. 扣除交易成本 (单边5bp)
```

## 5. 偏差处理

| 偏差类型 | 处理方式 |
|---------|---------|
| 生存偏差 | 不适用 (标准债券指数) |
| 前瞻偏差 | 不适用 (纯被动策略，无信号) |
| 选择偏差 | 低 (选择标准化债券指数ETF，非个券) |
| 交易成本 | 含单边5bp |

## 6. 适用场景

- **资金定位**: 防守底仓，低风险资金存放
- **适合投资者**: 风险厌恶型，追求稳定收益
- **互补策略**: 与策略B/C/E等进攻型策略搭配使用

## 7. 局限性

1. 收益率有限（约4%），无法跑赢通胀+机会成本
2. 债券熊市（利率快速上行）时仍有回撤
3. 信用债品种存在信用风险（虽然ETF已分散化）
4. 回测使用指数数据，实际ETF可能有跟踪误差
