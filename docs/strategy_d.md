# Strategy D: A股龙头股动量策略

> 28只行业龙头 · 动量选股 · 行业分散 · 双周调仓

## 1. 策略概述

**核心思想**：从28只覆盖15个行业的A股龙头股中，用4周动量（跳过最近1周避免短期反转）选出Top 8，等权持有，每行业最多1只确保分散。

| 维度 | 内容 |
|------|------|
| 标的 | 28只A股行业龙头股 |
| 信号频率 | 双周（每两周六运行） |
| 选股逻辑 | 4周动量 + Skip 1周反转保护 + Top8等权 |
| 行业约束 | sector_max=1（每行业最多1只） |
| 交易成本 | 单边8bp（万3佣金+千1印花税折算） |
| 无前瞻偏差 | T周决策 → T+1周收益 |

## 3. 策略逻辑详解

### 3.1 动量计算（带Skip）

```
动量 = Price[t-1] / Price[t-1-4] - 1
                 ↑skip=1   ↑lookback=4
```

- **Lookback = 4周**：回看过去4周的涨幅
- **Skip = 1周**：跳过最近1周的价格，避免短期均值回归效应

**为什么Skip 1？**
实证发现A股龙头股在最近1周有短期反转倾向（涨太多的短期回调），跳过它能提升动量信号的持续性。

### 3.2 行业约束（sector_max=1）

从动量排名中自上而下选择，每个行业最多入选1只：

```python
for stock in ranked_by_momentum:
    if count[stock.sector] < 1 and len(selected) < 8:
        selected.append(stock)
        count[stock.sector] += 1
```

**为什么这很重要？**
这是优化过程中**最大的单一改进**——避免行业集中风险（如同时持有茅台+五粮液），显著降低最大回撤。

### 3.3 调仓频率

每2周调仓一次（双周），平衡信号新鲜度与交易成本。

### 3.4 等权持有

选出的8只股票等权持有，每只12.5%。

## 5. 优化历程

策略经历了5轮系统优化：

| 轮次 | 内容 | 结果 |
|------|------|------|
| Round 1 | 基础策略开发，48+72组参数网格搜索 | 初步确定参数范围 |
| Round 2 | 深度优化：Vol缩放/Regime/行业约束/Walk-forward | **sector_max=1** 是最大改进 |
| Round 3 | 精细网格搜索（540 configs for D） | 微调最优参数 |
| Round 4 | Walk-forward 5折交叉验证 | 全部正收益 |
| Round 5 | 参数灵敏度分析 | 确认无悬崖效应 |

**关键发现**：
1. `sector_max=1` 是最大单一改进（Calmar显著提升）
2. Vol scaling和Regime filter对个股策略**无效**（只对ETF有效）
3. LB=4, Skip=1是最优组合

## 8. 注意事项

1. **无前瞻偏差**：严格T周决策→T+1周收益
2. **行业约束是核心**：去掉`sector_max=1`回测表现会大幅下降
3. **数据需要复权价**：使用Yahoo Finance的adjclose，已包含除权除息调整
4. **Yahoo限流**：28只股票下载需约2分钟（每只间隔3秒），内置429重试
5. **与策略E互补**：D是动量（买强势股），E是逆向价值（买被低估股），两者零重叠
