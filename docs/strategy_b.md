# Strategy B: 增强版 A股因子ETF 周度轮动

> 4因子动量轮动 + 双Regime择时 + 波动率缩放

## 1. 策略概述

**核心思想**：从4个低相关的Smart Beta因子ETF中，用动量排名动态分配权重，再叠加市场状态（Regime）和波动率缩放控制整体仓位，实现"牛市进攻、熊市防守"。

| 维度 | 内容 |
|------|------|
| 标的 | 4只A股因子ETF（现金流、国信价值、红利低波、创业成长） |
| 信号频率 | 每周（周五收盘后运行，下周一执行） |
| 选基逻辑 | 4周因子动量排名 → 50/30/20/0 权重分配 |
| 仓位控制 | 双均线Regime × 波动率缩放（目标15%年化波动率） |
| 交易成本 | 单边10bp（含佣金+冲击成本） |
| 无前瞻偏差 | T-1数据决策 → T期收益 |

## 2. 因子池

| 全收益指数 | 指数名称 | 对应ETF | ETF名称 | 定位 |
|-----------|---------|---------|---------|------|
| 932365CNY010 | 现金流 | 159201 | 万家自由现金流ETF | 防守 - 自由现金流筛选 |
| H21052 | 国信价值 | 512040 | 国信价值ETF | 防守 - 深度价值 |
| H20269 | 红利低波 | 515080 | 红利低波ETF | 防守 - 高股息低波动 |
| H00958 | 创业成长 | 159967 | 创业板成长ETF | 进攻 - 成长弹性 |

**设计理念**：前3个防守因子互相高相关（市场下行时都能扛），创业成长是进攻因子（与防守低相关），牛市时提供弹性。动量轮动自动在防守与进攻之间切换。

> 使用全收益指数（Total Return Index, 含分红再投资），ETF本身就包含分红，全收益指数更贴近实盘。

## 3. 策略逻辑详解

### 3.1 因子动量排名

每周计算过去**4周涨幅**（v5优化：原8周→4周，更快响应动量变化），按涨幅从高到低排名：

| 排名 | 权重 | 说明 |
|------|------|------|
| 第1名（最强） | 50% | 集中最强因子 |
| 第2名 | 30% | |
| 第3名 | 20% | |
| 第4名（最弱） | 0% | v5改进：完全不持有最弱因子 |

### 3.2 双时间框架Regime（中证全指 000985）

| 市场状态 | 条件 | 仓位系数 |
|---------|------|---------|
| 强牛 🟢 | 价格 > 13周MA 且 > 40周MA | 100% |
| 过渡 🟡 | 价格在两条MA之间 | 70% |
| 熊市 🔴 | 价格 < 13周MA 且 < 40周MA | 35% |

### 3.3 波动率缩放

```
Vol缩放系数 = min(目标波动率 / 实际波动率, 1.3)
```

- 目标年化波动率：15%
- 回看窗口：12周
- v6改进：上限从1.0提升至1.3x，允许低波时适度加杠杆

### 3.4 最终仓位计算

```
总仓位 = Regime系数 × Vol缩放系数
某因子仓位 = 总仓位 × 该因子动量权重
现金仓位 = 1 - 总仓位
```

## 7. 注意事项

1. **无前瞻偏差**：回测严格使用T-1数据决策、T期收益
2. **选择偏差**：4个因子是基于历史表现筛选的，存在survivorship/selection bias，未来表现可能不同
3. **数据降级**：CSIndex API偶尔被WAF封锁，自动降级使用东财价格指数
4. **实盘差异**：ETF买卖存在约0.1-0.15%的冲击成本，回测10bp可能略低估
5. **不构成投资建议**：本策略仅供研究参考
