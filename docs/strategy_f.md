# Strategy F: 美股QDII动量轮动策略

> 纳指100 vs 标普500 动量择优 + MA趋势防守 + 回撤锁定

## 1. 策略概述

**核心思想**: 在人民币计价的美股QDII ETF中做动量轮动，同时用均线系统进行趋势防守。当美股进入下行趋势时，自动切换到短债ETF避险。

| 维度 | 内容 |
|------|------|
| 策略类型 | 动量轮动 + 趋势择时 |
| 标的 | 2只美股QDII ETF + 1只债券避险 |
| 调仓频率 | 每周 (周五) |
| 配置方式 | Top1持100% |
| 交易成本 | 单边15bp (含QDII溢价) |
| 数据源 | 中证指数公司 (CSIndex API) |

## 2. 标的池

| 指数 | ETF | 代码 | 定位 |
|------|-----|------|------|
| 中证纳斯达克100 (H30533) | 广发纳指100ETF | 159941 | 美股科技成长 |
| 中证标普500 (H30140) | 博时标普500ETF | 513500 | 美股宽基价值 |
| 中证短债 (H11006) | 海富通短融ETF | 511360 | 避险/现金替代 |

**回测数据**: 使用中证编制的人民币计价指数（已含汇率波动），非美元原始数据。

## 3. 策略逻辑

```
每周五收盘:

  1. 计算纳指100和标普500的4周收益率
  2. 防守判断: 纳指100收盘价 < 20周MA?
     - 是 -> 全仓短债ETF (避险模式)
     - 否 -> 选4周动量最强的指数，100%持仓
  3. 回撤锁定: 若近期最大回撤 > -8% -> 锁定8周不操作
  4. T-1数据决策，T期收益，无前瞻偏差
```

### 核心参数

| 参数 | 值 | 说明 |
|------|---|------|
| momentum_lookback | 4周 | 动量计算窗口 |
| ma_fast | 8周 | 快速均线 (未直接用于信号) |
| ma_slow | 20周 | 慢速均线 (防守判断) |
| dd_limit | -8% | 回撤触发阈值 |
| dd_lockout | 8周 | 回撤锁定期 |
| txn_cost | 15bp | 单边交易成本 |

## 5. 偏差处理

| 偏差类型 | 处理方式 |
|---------|---------|
| 前瞻偏差 | T-1数据决策，T期收益 |
| 汇率偏差 | 使用人民币计价指数，已含汇率波动 |
| 交易成本 | 单边15bp (较高，含QDII溢价估算) |
| 选择偏差 | 低 (选择流动性最好的美股QDII ETF) |

## 6. 实盘注意事项

1. **QDII限额**: 实际交易中QDII ETF可能受额度限制，场内溢价可能很高
2. **折溢价风险**: QDII ETF折溢价波动可能显著影响实际收益，回测中未完全建模
3. **汇率风险**: 人民币计价隐含汇率波动，美元走强时有额外收益，反之亦然
4. **交易时差**: A股和美股交易时间不同，ETF价格可能滞后反映美股变化

## 7. 适用场景

- **资金定位**: 美股配置部分
- **互补策略**: 与A股策略（B/D/E/G）互补，提供地域分散化
- **适合投资者**: 需要海外配置但不便直接投资美股的投资者

## 8. 局限性

1. 15bp交易成本较高，侵蚀部分收益
2. QDII ETF折溢价风险未完全建模
3. 仅2只指数轮动，分散度有限
4. 回撤锁定机制参数（-8%, 8周）可能对不同市场环境不够自适应
