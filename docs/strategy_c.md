# Strategy C: 跨境ETF纯动量轮动

> Top1纯动量 · 恒生系ETF · 简洁有效

## 1. 策略概述

**核心思想**：在3只恒生系跨境ETF中，每周选4周动量最强的1只，100%持仓。无趋势过滤、无Regime择时、无波动率缩放 —— 极致简洁。

| 维度 | 内容 |
|------|------|
| 标的 | 3只A股上市的港股ETF |
| 信号频率 | 每周（周五收盘后运行，下周一执行） |
| 选基逻辑 | 4周动量排名 Top1，100%持仓 |
| 仓位控制 | 无（始终满仓） |
| 交易成本 | 单边8bp（跨境ETF无印花税） |
| 无前瞻偏差 | T周决策 → T+1周收益 |

## 2. 标的池

| ETF代码 | ETF名称 | Yahoo Ticker | 因子 |
|---------|---------|-------------|------|
| 513180 | 恒生科技ETF | 513180.SS | 科技 |
| 513060 | 恒生医疗ETF | 513060.SS | 医疗 |
| 159726 | 恒生高股息ETF | 159726.SZ | 高股息 |

**设计理念**：3只ETF覆盖港股三大板块（科技成长、医疗健康、高股息防守），彼此低相关。动量轮动自动追踪最强趋势。

## 3. 策略逻辑

### 非常简单的逻辑

1. 每周五收盘后，计算3只ETF过去4周涨幅
2. 选涨幅最高的1只，100%持有
3. 下周五再次评估，如有变化则换仓
4. 换仓时收取8bp×2（卖旧+买新）交易成本

```python
# 伪代码
momentum = price_now / price_4weeks_ago - 1
selected = max(etf_pool, key=momentum)
hold(selected, weight=100%)
```

### 为什么不加过滤？

- 标的池仅3只且均为港股ETF，加过滤器容易过拟合
- 回测期较短（~4年），参数越少越稳健
- 已在研究中对比过带过滤的方案，改善不显著

## 7. 注意事项

1. **回测期偏短**（~4年），统计意义有限，需谨慎对待回测结果
2. **无前瞻偏差**：严格T周决策→T+1周收益
3. **交易成本**：跨境ETF无印花税，仅佣金，按8bp计算
4. **独立运行**：策略C与策略B使用不同标的池，互不干扰
5. **港股波动大**：恒生系ETF波动性高于A股ETF，需注意风险
6. **Yahoo限流**：API有频率限制，脚本内置重试机制（429→等待重试）
