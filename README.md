# ETF短线交易监控系统

振威的A股ETF短线/中短线交易辅助系统。

## 策略概述

**本金**: 5万 CNY  
**品种限制**: 仅债券ETF + 中国宽基ETF + 行业ETF（无QDII）

### 资金分配

| 仓位 | 类型 | 品种 | 策略 |
|------|------|------|------|
| 1万 | 债券ETF T+0 | 511090/511010 | 跌0.1%买，涨0.15%卖 |
| 2万 | 宽基ETF中短线 | 510300/510500/588000/159915 | 大盘跌1.5%建仓，涨2%止盈 |
| 2万 | 行业ETF超跌 | 512480/516110/515070/159869/512170/515000 | 单日跌3%买，次日涨0.8%卖 |

### 风控规则
- ⚠️ 单日亏损 > 500元 → 停止操作
- ⚠️ 债券ETF止损: -0.25%
- ⚠️ 宽基ETF止损: -3%  
- ⚠️ 行业ETF止损: -4%

## 安装

```bash
pip install requests
```

## 使用

```bash
# 生成早盘操作建议 (9:00前运行)
python etf_monitor.py morning

# 生成收盘复盘 (15:15后运行)
python etf_monitor.py evening

# 实时监控模式 (交易时间内自动刷新)
python etf_monitor.py monitor
```
