> **自动化程度**：每日定时运行 · ADB自动下单 · Telegram播报

一套完整的A股ETF量化策略体系，包含：
- **宏观定仓**：PE估值百分位 × 市场趋势强度 9×9矩阵
- **行业轮动 V5**：动量确认 + 大盘/行业双重趋势过滤（最新版本）

## 📊 Strategy B：增强版因子ETF周度信号 (推荐)

> **最新版本 v3** | 4因子动量轮动 + 双Regime择时 + 波动率缩放

### 策略概述

每周五收盘后运行，输出下周调仓建议。核心思路：用低相关的Smart Beta因子分散风险，动量排名动态调权，双均线判断市场状态控制仓位，波动率缩放控制风险。

### 核心逻辑

**1. 因子动量排名（每周更新）**

计算每个因子过去8周涨幅，按涨幅排名分配权重：

| 排名 | 权重 |
|------|------|
| 第1名（最强） | 40% |
| 第2名 | 30% |
| 第3名 | 20% |
| 第4名（最弱） | 10% |

**2. 双时间框架Regime（中证全指 000985）**

| 市场状态 | 条件 | 仓位系数 |
|---------|------|---------|
| 强牛 | 价格 > 13周MA 且 > 40周MA | 100% |
| 过渡 | 价格在两条MA之间 | 70% |
| 熊市 | 价格 < 13周MA 且 < 40周MA | 35% |

**3. 波动率缩放**

- 目标年化波动率：15%
- 回看窗口：12周
- 缩放系数 = min(15% / 实际波动率, 1.0)

**4. 最终仓位**

```
总仓位 = Regime系数 x Vol缩放系数
每个因子仓位 = 总仓位 x 该因子动量权重
```

## 📊 核心策略：二维坐标系宏观定仓

策略核心：**市场估值（便宜还是贵）× 市场趋势（牛还是熊）共同决定仓位**。

### 仓位决策矩阵（9×9）

```
        ←——— 趋势强度 (T轴) ———→
        T1    T2    T3    T4    T5    T6    T7    T8    T9
        极熊  深熊  弱熊  偏熊  猴市  偏牛  弱牛  强牛  极牛
      ┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
  L1  │ 95% │ 90% │ 85% │ 80% │ 75% │ 65% │ 55% │ 45% │ 35% │ ← 极度低估(<10%位)
  L2  │ 90% │ 85% │ 80% │ 75% │ 70% │ 60% │ 50% │ 40% │ 30% │ ← 低估
  L3  │ 85% │ 80% │ 75% │ 70% │ 65% │ 55% │ 45% │ 35% │ 25% │ ← 偏低估
  L4  │ 80% │ 75% │ 70% │ 65% │ 60% │ 50% │ 40% │ 30% │ 20% │ ← 中性偏低
  L5  │ 70% │ 65% │ 60% │ 55% │ 50% │ 40% │ 30% │ 25% │ 15% │ ← 中性
  L6  │ 60% │ 55% │ 50% │ 45% │ 40% │ 30% │ 25% │ 20% │ 10% │ ← 中性偏高
  L7  │ 50% │ 45% │ 40% │ 35% │ 30% │ 25% │ 20% │ 15% │ 10% │ ← 偏高估
  L8  │ 40% │ 35% │ 30% │ 25% │ 35% │ 20% │ 15% │ 10% │  5% │ ← 高估
  L9  │ 30% │ 25% │ 20% │ 15% │ 10% │ 10% │  8% │  5% │  3% │ ← 极度高估(>88%位)
      └─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘
```

| 场景 | 估值 | 趋势 | 目标仓位 |
|------|------|------|---------|
| 🟢 超跌抄底 | 极低估(L1-L2) | 熊市(T1-T3) | 85-95% |
| 🟢 最佳买入 | 低估(L1-L3) | 牛市(T7-T9) | 55-95% |
| 🟡 均衡配置 | 中性(L4-L6) | 猴市(T5) | 40-60% |
| 🔴 高危减仓 | 极高估(L9) | 牛市(T7-T9) | 3-8% |

## 🔄 行业轮动策略（V3 → V5 演进）

### 策略演进与回测结果

> 回测区间：2013-06 ~ 2026-02（12.7年）
> 数据：申万22个一级行业，月度PE + 价格，历史175个月

| 版本 | 年化收益 | 最大回撤 | Sharpe | 核心改进 |
|------|---------|---------|--------|---------|
| V3 基准 | 10.8% | -50.5% | 0.63 | 低PE + 动量评分，历史PE缓存 |
| V4A 动量+大盘MA3 | 10.6% | -39.6% | **0.71** ✅ | 大盘等权MA3趋势过滤 |
| V4B 逆向+大盘MA3 | 0.9% | -44.8% | 0.14 ❌ | 逆向评分（已证明失效）|
| **V5A 动量+行业MA3** | **10.0%** | **-32.1%** | 0.65 | 行业级别趋势过滤 |
| 等权买持（参考） | 12.1% | -29.6% | 0.72 | — |

**V5A vs V4A 权衡**：
- MDD 改善 7.5%（-39.6% → -32.1%）✅
- 年化牺牲 0.6%（10.6% → 10.0%）
- 超额年化 -1.5% → -0.2%（接近等权买持）
- 推荐：回撤敏感用 V5A，追求Sharpe用 V4A

### 关键结论

1. **动量确认 >> 逆向视角**：年化 10.6% vs 0.9%，逆向策略是价值陷阱
   - 行业轮动需要催化剂，低PE + 上涨趋势 = 强信号
   - 低PE + 继续下跌 = 政策/基本面恶化，不是机会

2. **趋势过滤有效**：MA3大盘过滤将 MDD 从 -50.5% 压至 -39.6%（改善11%），Sharpe 0.63→0.71

3. **宏观止损(PE>90%清仓)有害**：策略专选低估行业，清仓会错过便宜板块的收益
   - 替代方案：PE>70% 时自动缩减至 Top2 行业（已内置）

4. **已修复前瞻偏差 Bug**：`calc_pe_percentile` 改用 `hist.iloc[:-1]` 排除当前月

### V5A 评分公式（动量确认视角）

```
综合分 = PE权重 × (100 - PE百分位) + 动量权重 × 动量分 + 共振奖励

PE权重 / 动量权重:
  宏观PE > 70%  → 62% / 38%  (高估期更看重低估)
  宏观PE < 35%  → 35% / 65%  (低估期更看重动量)
  其他          → 48% / 52%

动量分 = 0.55 × 3M动量得分 + 0.45 × 6M动量得分  (同向时+3分奖励)
行业过滤: PE百分位 > 85% → 硬排除
行业趋势: 价格 < 自身MA3 → 排除 (V5新增)
大盘趋势: 等权指数 < MA3 → 整体仓位 × 50%
```

### 宏观联动 Top-N

| 宏观PE百分位 | 持仓行业数 | 含义 |
|------------|----------|------|
| > 70% | Top 2 | 高估期精选 |
| 35-70% | Top 3 | 正常配置 |
| < 35% | Top 5 | 低估期广布 |
